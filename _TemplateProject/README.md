Шаблон проекта
==============

Шаблон для добавления проекта. В первом параграфе - описание. Текущий
текст удалить (шаблонный).

Структура:
src - исходники
  forms - используемые окна, формы
    frames - Фреймы 
    widgets - Qt виджеты
include - выходные .h
dependeces - включения внешниях файлов
