Лабораторная работа по теме
Программирование алгоритмов формирования и обработки двумерных 
массивов

Задание:
Сформируйте произвольные двумерные массивы X(m, n) и Y(n, k).
Получите массив Z путем прибавления к элементам каждого столбца
массива X произведения элементов соответствующей строки массива
Y.

Выполнено в VS Studio 2019
